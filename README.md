# gaokao
高考志愿参考系统

根据近三年的高考分数段统计表和某学校分专业分数线统计表，计算报考各专业的录取概率。

## 使用方法
1. 用自己省份的数据替换`data/`目录中的数据。
2. 修改`config.py`的内容。
3. 运行`gaokao.py`

    'firstLine':500,
    'score':588,
    'rank':4727
}
